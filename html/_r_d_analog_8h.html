void RDAnalogReadCont ( unsigned char  channel,
unsigned char  enable 
)

Enables/Disables auto trigger source in free running mode. In this mode the ADC will constantly sample and update the ADC register.

Parameters
channelThe pin that should be read (0 - 7)
enableCan be either ON or OFF. if enable == ON, the ADC will be enabled in free running mode. if enable == OFF, auto trigger will be disabled and the ADC will stop.
uint16_t RDAnalogReadToPerc ( unsigned char  channel,
unsigned char  mode,
uint32_t  multiplier 
)

Reads an analog signal and scales the returned value to a value from 0 to multiplier.

Parameters
channelThe pin that should be read (0 - 7).
modeThe mode can be MODE_8_BIT or MODE_10_BIT if mode == MODE_8_BIT, the resolution will be 256. if mode == MODE_10_BIT, the resolution will be 1024.
multiplierThe maximum value you want to scale the read value to, Eg. multiplier of 100 will return a value from 0 to 100.
Returns
Scaled representation of analog signal.
